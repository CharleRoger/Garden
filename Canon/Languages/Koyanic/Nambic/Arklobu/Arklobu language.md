#language/koyanic #krobu

**Arklobu** is a [[Koyanic languages|Koyanic language]] which was the predominant language in [[Krobu]] during the [[Krobu classic period]].

# Name

*[[Arklobu dictionary#arklobu|Arklobu]]* is derived from the name *[[krobu]]*, which itself means "melon land", via an initial element ostensibly of the original form *al-*, which has an unclear meaning and only appears in this term. The exchange in liquid consonants such that the first is rhotic and the second is lateral is a regular sound change which occurred in the development of the Arklobu language, implying the [[Old Arklobu]] form was *alkrobu*.

It has been suggested that the name originated as an [[Upic languages|Upic]] exonym due to the similarity of the initial syllable with the reconstructed [[Proto-Upic language|Proto-Upic]] root [[Proto-Upic dictionary#*ʔar|*ʔar]] "language" and especially its later [[Öbic languages|Öbic]] reflex \*al, which was used to form the name of the language *[[Low Alöbi|alöbi]]* in an apparently analogous way. The term *[[Arklobu dictionary#arklobu|arklobu]]*, however, referred not only to the language but also the speakers of the language and more broadly as a demonym for all inhabitants of [[Krobu]]. This latter sense was later supplanted by the collective noun *[[Arklobu peoples|arklobota]]*.

All vowels following the penultimate vowel in the word match the backness of that vowel:

| Front | Back |
| ----- | ---- |

| ü     | u    |
| e     | ë    |
| ö     | o    |
| ä     | a    |

# Grammar

## Nouns

### Number

Nouns fall into one of two categories according to their countability. Countable nouns express two grammatical numbers, singular and plural, which each have different morphological forms, while uncountable nouns have only a single form and cannot be modified by quantifiers. Uncountable nouns are typically mass nouns like materials and something, as well as abstract concepts and certain unique nouns like the sun and the world.

### Case

|            | -   | Singular | Plural    | Uncountable |
| ---------- | --- | -------- | --------- | ----------- |
| Absolutive | -   | *-(a)*   | *-(ë)ta*  | *-(ë)*      |
| Accusative | -   | *-(a)r*  | *-ota*    | *-u*        |
| Relative   | -   | *-(r)ä*  | *-(r)etä* | *-(r)i*     |

The relative case triggers certain stem-final consonants to mutate according to the following rules:

- {*r*, *l*, v, *h*} > *y*
- *b d g* > *p t ç*

### Ablaut

The final vowel in the stem of a noun follows one of four ablaut patterns, shown in the table below.

| Number   | Case     | -   | I     | II    | III   | <      | IV    | <      | <     |
| -------- | -------- | --- | ----- | ----- | ----- | ------ | ----- | ------ | ----- |
| Singular | Abs, Acc | -   | *-a-* | *-a-* | *-a-* | *-äi-* | *-a-* | *-öü-* | *-au* |
| ^        | Relative | -   | *-ä-* | *-ä-* | *-ä-* | ^      | *-ä-* | ^      | ^     |
| Plural   | Abs, Acc | -   | *-a-* | *-ë-* | *-i-* | <      | *-u-* | <      | <     |
| ^        | Relative | -   | *-e-* | *-i-* | ^     | <      | *-ü-* | <      | <     |

## Adverbs

Arklobu features a class of adverbs derived from the lexicalisation of an older ablative case. They are often transparently related to nouns, though can show irregularities and semantic differences.

| Adverb                             | Noun                                       |
| ---------------------------------- | ------------------------------------------ |
| *ñüpi* "manually"                  | *ñuba* "hands.ABS"<br>*ñüpetä* "hands.REL" |
| *kmüfi* "sincerely"                | *kmof* "flesh.ABS"<br>*kmüfi* "flesh.REL"  |
| *öñçidi* "with assistance"         | *öñçöd* "help.ABS"<br>*öñçedi* "help.REL"  |
| *fini* "correctly, conventionally" | *fäir* "road.ABS"<br>*fäiri* "road.REL"    |
| *müdi* "every day, often, always"  | *möye* "day.ABS"<br>*müçi* "day.REL"       |
| *hä* "reflexively"                 | *hahala* "self.ABS"<br>*hoherä* "self.REL" |

## Verbs

### Nonfinite verbs

Two nonfinite verb forms exist, the **gerund** and the **infinitive**.

The gerund serves as the citation form of the verb, despite being the only form which is not actually a verb. The declension of the gerund is shown in the table below.

|            | -   | Ending |
| ---------- | --- | ------ |
| Absolutive | -   | *-nor* |
| Accusative | -   | *-nau* |
| Relative   | -   | *-näï* |

The infinitive is formed with the suffix *-(ç)i* attached to the mutated stem. The presence of *-ç-* is determined by the final segment of the stem, as shown in the table below.

| Mutated stem | Infinitive |
| ------------ | ---------- |
| *-p*         | *-pç*      |
| *-t*         | *-t*       |
| *-f, -h, -ç* | *-ç*       |
| *-m*         | *-mç*      |
| *-n, -ñ, -ŋ* | *-ñç*      |
| *-ä*         | *-äi*      |
| *-i*         | *-içi*     |
| *-ü*         | *-öü*      |
